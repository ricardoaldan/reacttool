
## Prerequisites

Install [Node.js and npm](https://nodejs.org/en/) (v6.6.0 or newer) if they are not already
installed on your computer.

## Steps to build it

1.	Npm install webpack -g
2.	Npm install tsd -g
3.	Npm install
4.	Npm start

- Then, load http://localhost:8080/ in a browser to see the output.
